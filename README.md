# Dehashed Site
based on [dehashed](https://dehashed.com)

You can find my hosted site at https://dehashed.cyberthing.dev

## Installation

```bash
git clone https://github.com/purplelemons-dev/dehashed-site.git
cd dehashed-site
npm i
```

## Run

```bash
npm run dev2
```

## Build Container
After install,

```bash
docker compose up --build
```
